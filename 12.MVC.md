# MVC



M - Model (数据模型) 负责操作所有数据
V - View (视图) 负责所有 UI 界面
C - Controller (控制器) 负责其他



# 添加 jQuery 的方式

## 用 yarn 添加



```shell
yarn add jquery
```

## 使用



```ts
import $ from 'jquery'
import './app2.css'
//...
```



# 在 JS 中 import css



```ts
// 在 app1.js 中

import './app2.css'
```



# localStorage



```ts

const localKey = 'app2-ind';
const ind = localStorage.getItem(localKey) || 0;

localStorage.setItem(localKey, ind);

```
